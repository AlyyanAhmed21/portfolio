# Neural Grid Cursor Reveal with Auto Fade

A Pen created on CodePen.

Original URL: [https://codepen.io/gauravgajjar/pen/azvvWPx](https://codepen.io/gauravgajjar/pen/azvvWPx).

A futuristic interactive background effect where glowing gradient-bordered squares reveal and fade dynamically as the user moves the cursor. Inspired by neural pathways and data grids, this effect responds to mouse movement with smooth randomized box transitions and fading trails. Ideal for tech websites, portfolio intros, or digital agency landing pages.

Includes:

Mouse-tracked square reveal

Radial gradient borders (no fill)

Smooth fade-out after reveal

Performance-optimized canvas animation

Fixed headline prompt: "Move Your Cursor"

Fully responsive, lightweight, and perfect for adding immersive depth to modern UIs.
